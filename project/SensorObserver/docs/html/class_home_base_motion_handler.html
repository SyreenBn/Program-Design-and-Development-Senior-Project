Detailed Description

The handler for the home base's actuators, which in this case are the two wheel actuators.

HomeBaseMotionHandler manages the modification to the velocity based on collisions. HomeBaseMotionHandler translates velocity and position to a new position. Both of these are straightforward, but the framework allows for more sophisticated models of motion in which each wheel has distinct speed.

For this iteration, both wheels are always going at maximum speed, and cannot be controlled independently.
